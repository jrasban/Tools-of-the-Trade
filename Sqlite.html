What is SQLite?

SQLite is a database management software that does not require configuration. Unlike MySQL and Microsoft SQL Server,SQLite takes very little time to get set up. This lightweight version of SQL is fast and easy to use. You can use the command line to manage your SQLite databases or you can download the free open source SQLite Studio to graphically interact with the data. Python connnects very easily with Sqlite so you can manipulate, add, and store new data right from python. SQLite is not always the best choice as a database management solution, but it certainly does a lot of great work!
